# AheadAI – Smart Task Planner
AheadAI is an AI-powered productivity web app that helps users break down tasks, generate structured plans, and improve focus using intelligent suggestions.

## Features
- 🧠 AI-powered task planning
- 📊 Productivity score system
- 🎯 Smart suggestion chips
- ⏱️ Focus-oriented task breakdown
- 🎨 Modern glassmorphism UI
- 🌌 Animated landing page with glowing effects
- 📱 Fully responsive layout

## Tech Stack
- React.js (Vite)
- JavaScript (ES6)
- CSS (Glassmorphism + Animations)
- AI simulation logic (custom API function)
